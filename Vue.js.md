# vue.js

설치


1. cdn 
2. npm
3. vue-cli
```
// vue-cli 전역 설치, 권한에러시 sudo 추가
npm install -g vue-cli

// vue init <template-name> <project-name>
vue init webpack my-project

cd my-project

// npm 패키지 설치
npm install

npm run dev

localhost:8080
```
